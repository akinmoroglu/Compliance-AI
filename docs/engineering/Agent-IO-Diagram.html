2
Request Context Store
In-memory, per-request — not persisted to DB

Fields stored in Go request context and passed between phases

image_bytes — raw image bytes or nil. Written by Pre-Processor. Read by Vision Agent and Alignment Agent.
video_frames[] — slice of frame byte slices extracted by ffmpeg. Written by Pre-Processor. Read by Vision Agent.
audio_bytes — .mp3 audio track bytes. Written by Pre-Processor. Read by Audio Agent.
audio_transcript — verbatim transcript. Written by Audio Agent after Phase 1. Read by Violations Agent (Phase 1b), Personal Attribute Agent (Phase 1b), and Alignment Agent (Phase 2).
image_extracted_text — OCR output from Vision Agent. Written by Vision Agent after Phase 1. Read by Violations Agent (Phase 1b).
lp_text_excerpt — structured LP text (body_start + body_end + headings). Written by LP Agent. Read by Alignment Agent.
lp_key_claims — structured claim set from LP Agent. Written by LP Agent. Read by Alignment Agent.
category — classification output. Written by Category Classifier. Read by all Phase 1 agents.
