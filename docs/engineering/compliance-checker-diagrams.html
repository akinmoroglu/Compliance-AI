2
Scoring Engine — Decision Waterfall
First matching rule wins. Pure deterministic software — no LLM involved. Takes the violations list from Phase 2 agents as input.
Rule 1 — Prohibited violation
Any P_ code in violations list?
if p_codes: → immediate exit
score: 0 High Risk not_compliant No fix possible
no P_ → continue
Rule 2 — Unauthorized authorization-required category
Any A_ code AND advertiser not authorized?
if a_codes and not authorized: → immediate exit
score: 0 High Risk not_compliant Shows authorization path
no unauthorized A_ → continue
Rule 3 — Restricted content violations
Any R_ codes present? (e.g. weight-loss claim, body shaming, health claim...)
base = 50
− 10 × (each additional R_ after the first)
− 5 × (each M_ stacked on top)
floor = 1
score: 1–50 Medium or High Risk compliant Fix shown per violation
Examples
1 R_ only → 50 2 R_ → 40 3 R_ → 30 2 R_ + 1 M_ → 35 1 R_ + 3 M_ → 35
no R_ → continue
Rule 4 — Quality / deception issues only
M_ codes only (no P_, A_, or R_)? (e.g. urgency, narrative, hygiene, sensationalism...)
base = 75
− 5 × (each additional M_ after the first)
floor = 45
score: 45–75 Low or Medium Risk compliant Fix shown per violation
Examples
1 M_ → 75 2 M_ → 70 3 M_ → 65 6 M_ → 50 7+ M_ → 45 (floor)
no violations → continue
Rule 5 — Clean ad, standard category
No violations + category is ALLOWED
score = 95 (range: 85–100)
score: 85–100 Low Risk compliant
Rule 6 — Clean ad, sensitive category
No violations + category is RESTRICTED or authorized AUTH_REQUIRED
score = 78 (range: 71–84)
score: 71–84 Low Risk compliant
